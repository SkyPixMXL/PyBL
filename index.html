PyBL

AI development with the use of webcam

PyBL is a non-profit company that uses the webcam to deduce in real time the positions of joints such as those in the hand, body, etc. The tracking model is written entirely in JavaScript. The tracking model is entirely written in JavaScript and contains several parameters included in different models, which I hope will allow you to discover the beauty of coding. PyBL's different models run entirely in the browser, so no data need be sent to the server, although it does require the user's consent to access his webcam.


Hands Tracking Pose Estimation Iris Tracking

For the basics of the program, I took inspiration from the MediaPipe library, from which I perfected three models: hand joint estimation, body joint estimation and gaze redirection. The fact that they are coded in JavaScript makes them more accessible.

Usage


PyBL uses mainly the java script language to run the various models. The models offered on this website therefore require no installation and run on all types of device.

This website is still under development. Please let us know if you encounter any problems.

Once you chosen one of the models on this site, clicking on the try button will take you to another web page, where you'll be asked to authorize access to the camera.

Once you've accepted, just wait a few seconds and you'll see the camera return with the changes made by the chosen model.


Home

By clicking on the PyBl logo at the top left of the page, you can return to the home page.

Settings

Through the setting panel on the right-hand side of the page, you can explore other models, but still work with the basic principle of the models you've chosen.

Pause and Resume

Using the pause button on the right-hand side of the page, this stops the video and the camera switches off. To switch it back on, simply press the pause button again..
